# Machine Learning Paper

This folder contains the separate Machine Learning course paper.

- `machine_learning_paper.tex` is the LaTeX source.
- `machine_learning_paper.pdf` is the compiled paper.

This paper overlaps with the Current / Computing Trends final paper, but it
places more emphasis on machine learning setup, tokenization, model development,
hyperparameter choices, and metric interpretation.

Generated LaTeX files such as `.aux`, `.log`, and `.out` may remain here after
compilation. Shared images live in `../images/`.

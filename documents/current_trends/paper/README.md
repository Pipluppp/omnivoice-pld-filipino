# Paper

This folder contains the Current / Computing Trends written outputs.

- `research_proposal.tex` and `research_proposal.pdf` are the earlier proposal
  paper.
- `final_paper.tex` and `final_paper.pdf` are the final paper for the Current /
  Computing Trends course, written in ACM format (`acmart`, sigconf, two
  columns) with `references.bib` as its BibTeX bibliography.
- `final_paper.text` is a plain text extraction of an earlier (pre-ACM) draft
  kept for quick search and review.

Generated LaTeX files such as `.aux`, `.log`, and `.out` may also appear here
after compilation. Shared images live in `../../images/`.

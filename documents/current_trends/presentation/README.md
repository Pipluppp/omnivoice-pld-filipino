# Presentation

This folder contains the presentations for the Current / Computing Trends
course.

- `proposal_presentation.md` is the proposal slide content in markdown form.
- `proposal_beamer.tex` / `proposal_beamer.pdf` are the proposal deck.
- `final_beamer.tex` / `final_beamer.pdf` are the final project deck, covering
  the whole project: problem, background, dataset, pipeline, training runs,
  results, discussion, limitations, and conclusions.
- `beamerthemedurham.sty` is the local Beamer theme file used by both decks.

The proposal deck is proposal-stage material (research problem, model choice,
dataset, methodology, evaluation plan). The final deck reuses the same theme
but reports completed results. Shared images live in `../../images/`.

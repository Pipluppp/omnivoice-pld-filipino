# Presentation

This folder contains the proposal presentation for the Current / Computing
Trends course.

- `proposal_presentation.md` is the slide content in markdown form.
- `proposal_beamer.tex` is the Beamer source.
- `proposal_beamer.pdf` is the compiled proposal deck.
- `beamerthemedurham.sty` is the local Beamer theme file used by the deck.

The presentation is proposal-stage material, so it focuses on the research
problem, model choice, dataset, methodology, and evaluation plan. Shared images
live in `../../images/`.
